                    BODMAS BUS EXPRESS
                    ===================
        "Hop on the Math Bus! Learn BODMAS the Fun Way!"

📌 HOW TO RUN THE GAME:
=======================

1. Save all 4 files in the same folder:
   - index.html
   - game.html
   - style.css
   - game.js

2. Open "index.html" in any web browser (Chrome, Firefox, Edge, Safari)

3. Enter your name and choose a character

4. Click "START JOURNEY" to begin playing!


🎮 HOW TO PLAY:
===============

1. You'll see a mathematical expression on screen
2. Click which operation to solve FIRST according to BODMAS rule
3. Correct answers move the bus forward and earn points
4. Wrong answers give you hints
5. Complete 5 expressions per level to advance
6. Reach the picnic to win!


📊 SCORING SYSTEM:
==================

✓ Correct answer: +10 points
✓ Streak bonus (3+ correct): +5 points
✓ Streak bonus (5+ correct): +10 points
✓ Perfect expression: +25 bonus
✓ Level complete: +100 bonus
✓ Game complete: +500 bonus


🏆 LEVELS:
==========

Level 1 - Beginner: Only × and ÷ (numbers 1-20)
Level 2 - Easy: All operations (numbers 1-50)
Level 3 - Medium: Round brackets ( ) (numbers 10-99)
Level 4 - Advanced: Square brackets [ ] (numbers 10-999)
Level 5 - Expert: Curly braces { } (numbers 100-9999)


💡 BODMAS RULE:
===============

B → Brackets first ( ) [ ] { }
O → Orders (powers)
D → Division ÷
M → Multiplication ×
A → Addition +
S → Subtraction -

Remember: DM and AS are solved LEFT to RIGHT!


🔧 TROUBLESHOOTING:
===================

❌ Game not loading? 
   → Make sure all 4 files are in the same folder
   → Check internet connection (for fonts)

❌ Score not saving?
   → Enable cookies/local storage in browser
   → Don't use incognito mode

❌ Buttons not working?
   → Refresh the page (F5)
   → Clear browser cache
